# 🚢 Titanic-Survival-Prediction

Hi there! 👋 This repository contains my approach to the classic **Titanic: Machine Learning from Disaster** competition. The goal of this project is to build a predictive model that answers the question: "What sorts of people were more likely to survive?"

As a statistics student, working on this famous dataset was a fundamental step in solidifying my data analysis and machine learning skills!

## 🎯 Project Overview
The Titanic dataset is a great starting point for binary classification problems. In this project, I explored passenger data (such as age, gender, ticket class, etc.), handled missing values, engineered new features, and trained various machine learning models to predict survival probabilities.

## 🛠️ Tech Stack & Tools
- **Language:** Python 🐍
- **Data Handling & EDA:** Pandas, NumPy
- **Data Visualization:** Matplotlib, Seaborn
- **Machine Learning:** Scikit-Learn
- **Environment:** Kaggle Notebook
